Shows two extra buttons (Scroll to top and Publish/Update) in the bottom of the screen when user scrolls near bottom.

== Description ==

1. Save posts/pages from bottom of the screen
2. Suitable for posts and pages with a lot of meta boxes, or when edit.php just tends to get very long.

* [Github](https://github.com/urre/update-from-bottom)

== Installation ==

1. Upload the `update-from-bottom` folder to your `/wp-content/plugins/` directory
2. Activate the "Update from Bottom" plugin in your WordPress administration interface
3. Done!
